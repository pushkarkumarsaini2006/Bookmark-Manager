# Bookmark Manager

A simple, single-page bookmark manager to organize and quickly access your favorite websites.

## Live Demo

[bookmark-manager-blue.vercel.app](https://bookmark-manager-blue.vercel.app/)

## Features

- **Real-time Search** - Filter bookmarks instantly as you type
- **Organized Categories** - Primary Bookmarks, Secondary Bookmarks, and Development Tools
- **Clean UI** - Modern gradient design with smooth hover animations
- **No Dependencies** - Pure HTML, CSS, and JavaScript

## Bookmarks Included

### Primary Bookmarks
- GitHub, Stack Overflow, MDN Web Docs, VS Code, Google

### Secondary Bookmarks
- CodePen, LeetCode, Dev.to, freeCodeCamp, Regex101, JSON Formatter, Docker Docs, Postman

### Development Tools
- Can I Use, NPM Registry, CDNJS, JWT Debugger, HTTP Status Codes, Base64 Decoder, Carbon, Ray.so

## Customization

Edit `index.html` to add or modify bookmarks:

```html
<li>
    <a class="bookmark-link" target="_blank" href="https://your-site.com">
        <span class="icon">🚀</span>Your Site
    </a>
</li>
```
